SDKError

public enum SDKError: Error

Errors that Qualaroo SDK can throw

  • invalidAPIKey: given apiKey is invalid. You can get it from https://app.qualaroo.com/dashboard by tapping on Setup when you expand selected site/application.
  • noSurvey: There is no active survey with given name or it was corrupted.
  • invalidSiteId: Paramether siteId should be non-empty String with only digits.
  • dataEncodingException: Problem occured while using data(using: String.Encoding) function.
  • languageCodeInvalid: Language code must be type of string with exactly two letters.
  • alreadySeenOrFinished: There is survey with that alias but user already seen it or finished it, and it shouldn’t be shown again or it’s to soon to display it.
  • noInternet: There is no access to web right now.
