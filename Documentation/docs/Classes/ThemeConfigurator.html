    backgroundColor

    Color used as background for all questions and messages.
    textColor

    Color of text used by all questions and messages.
    dimType

    UIBlurEffectStyle used under survey view. Apart from blur survey have alpha applied on this dimming view.
    buttonDisabledColor

    Color of disabled buttons, and initial No button.
    buttonEnabledColor

    Color of enabled buttons.
    buttonTextColor

    Color of text on buttons.
    borderColor

    Corder which is used only for border of text fields.
    tintColor

    Color used for x button that is closing survey, and for NPS segmented view.
    logo

    Image displayed as logo for surveys. If not provided, then qualaroo logo will be used.
